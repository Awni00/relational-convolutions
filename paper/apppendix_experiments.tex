\subsection{Relational Games}

\Cref{tab:relational_games_tasks} contains text descriptions of each task in the relational games dataset in the experiments of~\Cref{ssec:exp_relational_games}.~\Cref{tab:relgames_architectures} contains a description of the architectures of each model (or shared component) in the experiments.~\Cref{tab:relgames_group_architectures} contains a description of the hyperparameters of the grouping layers corresponding to the `Temporal G', `Feature G', and `Contextual G' RelConvNet models.
